# Project assets

## Project-page QR code

`pathosynvlm_project_page_qr.svg` and `pathosynvlm_project_page_qr.png` encode the canonical project website directly:

<https://atlasanalyticslab.github.io/PathoSynVLM/>

The code does not use a URL shortener, redirect service, or expiring token. QR codes themselves do not expire; it will continue to work as long as the project URL remains available. Use the SVG for print and the PNG for slide decks or other software that does not accept SVG.

Both versions use error-correction level H and a four-module quiet zone. If the project URL changes, regenerate both files together and verify the decoded value before committing them.
